    模仿一下组件UI文档咋来的，显示的按钮就是下面的vue文件代码。

    # 按钮


    <template>
    <div id='btn' @click="btnClick">
        <h-btn type='red' btnText='测试按钮1'/>
        <h-btn type='blue' btnText='测试按钮2'/>
        <h-btn type='yellow' btnText='测试按钮3'/>
        <h-btn type='green' btnText='测试按钮4'/>
    </div>
</template>

<script>
    export default {
        methods: {
            btnClick(e) {
                console.log(e.target.innerText)
            }
        }
    }
</script>

    # 导入代码段

    这里官网有点坑

    你可以通过下述的语法导入已经存在的文件中的代码段：

    <<< @/filepath
    1

    然后我就照做了，页面显示的是字符串。

    然后我以为是路径的问题，改了半天也没用，后来发现，是官方例子有问题。

    正确如下：没错！不要中间的空格！

    <<<@/docs/.vuepress/components/btn.vue
    1
